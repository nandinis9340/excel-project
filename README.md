# 📊 Microsoft Excel – Basic to Advanced Concepts

This repository demonstrates my practical knowledge of Microsoft Excel through hands-on examples and practice files. It covers essential Excel functions, data analysis techniques, and productivity tools commonly used in business, finance, and data analytics.

- Data Types

- Sorting & Filtering
- Conditional Formatting
- Data Validation

### 📊 Pivot Tables

- SUM
- AVERAGE
- COUNT & COUNTA
- IF

- COUNTIF / COUNTIFS
- MAX & MIN
- LEFT, RIGHT, MID

- TODAY & NOW
